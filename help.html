Quick Start

  1. Upload your data in csv format.
  2. Select a false discovery rate (FDR) value. These will be applied to multiple t-tests, using the Benjamini-Hochberg procedure. Typical values are between 0.05 and 0.3.
  3. Advanced Options: ensure that all the boxes are checked (unless you prefer not to use them). For more information on what these do, check out (references)
